# X Crypto

A PHP package for managing and creating token instances for various blockchains. This package provides a unified interface for interacting with different tokens (ERC20, BEP20, etc.) across various blockchain networks.

## Features

- Create token instances dynamically based on configuration for different blockchain types (`eth`, `bsc`, `polygon`, `tron`, etc.)
- Supports token-specific operations such as transferring tokens.
- Handles different token standards with ease by using interfaces for modular support.

## Installation

To install this package, you can use Composer. Run the following command in your project directory:

```text
composer require mitoop/x-crypto
```
